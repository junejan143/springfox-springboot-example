== Paths
=== 创建用户
----
POST /user
----

==== Parameters
[options="header"]
|===
|Type|Name|Description|Required|Schema|Default
|BodyParameter|user|user|true|<<User>>|
|===

==== Responses
[options="header"]
|===
|HTTP Code|Description|Schema
|201|Created|integer (int32)
|===

==== Consumes

* application/json

==== Produces

* */*

==== Tags

* User
